Warm matte surface container — the base for episode cards, quote blocks, panels.

```jsx
<Card accent>
  <Kicker>Episode 47</Kicker>
  <h3>The Discipline Behind the Drift</h3>
</Card>
```

Set `accent` for the short purple top tab. Override `padding` for tighter/looser cards. Matte shadow only — never add glow.
