Uppercase eyebrow label that sits above a title — echoes the logo's clock-tick dash.

```jsx
<Kicker tone="purple">New Episode · Every Tuesday</Kicker>
<Kicker tone="amber">Pause Prompt</Kicker>
```

Tones: `purple | amber | muted | ivory`. Set `dash={false}` to drop the leading tick. Keep it one short line.
