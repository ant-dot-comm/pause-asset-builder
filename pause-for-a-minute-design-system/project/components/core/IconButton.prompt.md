Circular single-glyph control — play, pause, share, etc. Always give a `label`.

```jsx
<IconButton variant="solid" size={56} label="Play episode">▶</IconButton>
<IconButton variant="soft" label="Share"><img src="assets/logos/pause-bars.svg" width="14" /></IconButton>
```

Variants: `solid` (purple, the play action), `soft` (tinted purple), `outline`. Pair with a Lucide glyph or the brand pause bars.
