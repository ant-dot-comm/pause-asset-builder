Small uppercase pill for status & metadata — episode numbers, "New", season, "Live".

```jsx
<Badge tone="purple">Episode 47</Badge>
<Badge tone="amber" solid>New</Badge>
<Badge tone="neutral">Season 3</Badge>
```

Tones: `purple | amber | neutral`. `solid` fills it; default is outline. Keep the text short — this is metadata, not a label paragraph.
