The brand symbol (clock arc + ticks + purple pause bars) as an inline, recolorable SVG.

```jsx
<PauseMark size={56} />
<PauseMark size={32} color="var(--ivory-50)" accent="var(--accent-soft)" />
<PauseMark size={28} arc={false} />  {/* pause bars only */}
```

Use as an icon, section accent, or quiet watermark. Keep `accent` on the bars purple in most contexts.
